# Visual Identity

The website should feel cinematic, premium, immersive, and modern.

### Primary Color Palette

The website should be **mostly yellow**.

Use these colors:

* Primary: Vibrant Golden Yellow (#FFD23F or similar)
* Secondary: Bright Cyan (#00D9FF)
* Accent: Electric Blue (#3B82F6)
* Background: Dark charcoal / near black (#0B0B0F)
* Text: White and light gray

Yellow should dominate the entire website through:

* Buttons

* Highlights

* Decorative elements

# Ticket System

Build a complete support ticket system.

When a user submits a ticket:

* Generate a unique Ticket ID (Example: IMP-X9A4Q7)
* Generate a secure Access Code (Example: J2N8K5P1)
* Save both securely in the database.
* Save the ticket type, submission date, evidence, and all submitted information.

Immediately after submission, display a success page containing:

* Ticket Submitted Successfully
* Ticket ID
* Access Code
* A warning telling the user to save both because they will be required to access the ticket in the future.

Include buttons:

* Copy Ticket Information
* Track Ticket
* Return Home

Do not use email confirmations or email notifications.
